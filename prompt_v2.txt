SYSTEM
You generate a single Python function that animates on a 2x20 ASCII VFD using DiffAnimator. Output only valid Python code with signature: def anim_TIMESTAMP_RAND(animator: DiffAnimator, duration: float = 10.0). No extra text, comments, or formatting.

Hardware contract
- Display: 2 rows x 20 columns; x in [0,19], y in {0,1}. ASCII only (no Unicode, box-drawing, block fills).
- Each frame: line1 = [' '] * 20; line2 = [' '] * 20; draw by index; then:
  animator.write_frame(''.join(line1), ''.join(line2))
  animator.frame_sleep(1.0 / animator.frame_rate)
- Duration: ~10s; frame_count = int(duration * animator.frame_rate)

Aesthetic: entity grouping
- One coherent entity with clear silhouette moving or evolving as a unit; every non-space belongs to entity footprint.
- No stray pixels, no detached flickers.
- Dual-row use only when part of entity silhouette, not separate random process.
- Character weight (pick ONE hierarchy): Heavy: O *; Medium: o + x ^ v < > |; Light: . - ` ,

Forbidden
- Base + random noise hybrids
- Unbounded trails or accumulation
- 2D particle physics (display has only two rows)
- Conflicting character meanings in same piece

EXAMPLE: Caterpillar with turnaround animation
When bouncing off walls, the hat marker moves through the body as it reverses, creating illusion of 3D rotation.

from cd5220 import DiffAnimator
import math

def anim_caterpillar_turnaround(animator: DiffAnimator, duration: float = 10.0):
    frame_count = int(duration * animator.frame_rate)
    body_len = 5
    pos_x = 5
    vel = 1
    hat_offset = 0
    turning = False
    turn_frames = 0
    
    for frame in range(frame_count):
        line1 = [' '] * 20
        line2 = [' '] * 20
        
        if not turning:
            pos_x += vel
            if pos_x >= 19 - body_len or pos_x <= 0:
                turning = True
                turn_frames = 0
        
        if turning:
            turn_frames += 1
            hat_offset = turn_frames
            if turn_frames >= body_len:
                turning = False
                hat_offset = 0
                vel = -vel
        
        if vel == 1:
            for i in range(body_len):
                if pos_x + i < 20:
                    line1[pos_x + i] = 'o'
            hat_pos = pos_x + body_len - 1 - hat_offset
            if 0 <= hat_pos < 20:
                line1[hat_pos] = '^'
        else:
            for i in range(body_len):
                if pos_x + i < 20:
                    line1[pos_x + i] = 'o'
            hat_pos = pos_x + hat_offset
            if 0 <= hat_pos < 20:
                line1[hat_pos] = '<'
        
        animator.write_frame(''.join(line1), ''.join(line2))
        animator.frame_sleep(1.0 / animator.frame_rate)

EXAMPLE: Breathing cluster
Symmetric dot field expands and contracts around center anchor.

from cd5220 import DiffAnimator
import math

def anim_breathing_cluster(animator: DiffAnimator, duration: float = 10.0):
    frame_count = int(duration * animator.frame_rate)
    cx = 10
    for frame in range(frame_count):
        line1 = [' '] * 20
        line2 = [' '] * 20
        r = 2 + int(3 * abs(math.sin(frame * 0.12)))
        for dx in range(r + 1):
            if cx - dx >= 0:
                line1[cx - dx] = '.'
            if cx + dx < 20:
                line1[cx + dx] = '.'
        line1[cx] = 'o'
        animator.write_frame(''.join(line1), ''.join(line2))
        animator.frame_sleep(1.0 / animator.frame_rate)

EXAMPLE: Mirrored pair
Two entities with symmetric oscillating motion.

from cd5220 import DiffAnimator
import math

def anim_mirrored_pair(animator: DiffAnimator, duration: float = 10.0):
    frame_count = int(duration * animator.frame_rate)
    left = 5
    right = 14
    for frame in range(frame_count):
        line1 = [' '] * 20
        line2 = [' '] * 20
        d = int(3 * math.sin(frame * 0.1))
        x1 = max(0, min(19, left + d))
        x2 = max(0, min(19, right - d))
        line1[x1] = '>'
        line2[x2] = '<'
        animator.write_frame(''.join(line1), ''.join(line2))
        animator.frame_sleep(1.0 / animator.frame_rate)
